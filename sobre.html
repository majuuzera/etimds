Saber mais sobre o curso

O técnico em desenvolvimento de sistemas forma profissionais capazes de desenvolver e programar sistemas computacionais, de acordo com normas e padrões de qualidade, integridade e segurança da informação. Também realiza a manutenção de programas de computador, análise de dados e linguagem de programação.

O futuro de quem escolhe programação

O futuro desse profissional é variado de acordo competências. Podendo trabalhar em casa e em escritório, ou até mesmo em outros países, por conta de sua alta demanda no mercado de trabalho. Tendo opções para diversos tipos de profissionais; back-end, front-end, hardware, banco de dados e muito mais.

Grade Curricular:



Análise Estruturada de Sistemas Estrutura de Dados Desenvolvimento de Software Seguro Programação e Design para Web Sistemas de Banco de Dados
Entre as matérias da grade curricular do curso de Análise e Desenvolvimento de Sistemas, está a Análise Estruturada de Sistemas. Aqui, você toma contato com o estudo de performance e os tipos de complexidade de algoritmo. Com ela, é possível fazer aplicações sólidas seguindo padrões de desenvolvimento.
Muito se fala na quantidade de informações que é gerada hoje na internet, principalmente por causa das redes sociais. Para você ter uma ideia, uma simples procura por vagas no LinkedIn já é o suficiente para diversos insights. Mas, para que eles se tornem, de fato, úteis aos negócios, precisam ser analisados e bem trabalhados. É então que surge a necessidade de compreender as estruturas de dados: desde como eles se organizam para a formação de um sistema (lineares e não lineares, por exemplo) até as possíveis arquiteturas usadas.
Sabe os mesmos dados de que falamos? Como você bem sabe, é preciso investir em segurança para evitar o vazamento de senhas e de informações pessoais dos usuários. Por isso, está também entre o que se estuda na Análise de Desenvolvimento de Sistemas. Você tem contato com noções de criptografia, detecção de possíveis fraudes ou intrusões, protocolos de gerenciamento e outras políticas de segurança para software.
A preocupação com o front-end das aplicações também se encontra nas disciplinas de Análise e Desenvolvimento de Sistemas. Algumas linguagens de programação entram aqui também, entre as quais podemos citar: JavaScript, HTML e CSS. Ainda há o aprendizado na construção de mocapes e provas de conceito de layout, com o foco na experiência do usuário com o produto final.
Na lista do que faz um analista de sistemas, não podia faltar o sistema de gerenciamento de banco de dados. Nessa disciplina, você vai conhecer os principais do mercado, como:
  • Oracle;
  • MySQL;
  • DB2;
  • SQL Server.
É possível aprender não apenas os conceitos, mas o que cada um desses sistemas pode fazer em termos de dados, de hardware, de usuários e de software.
